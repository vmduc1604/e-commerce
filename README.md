# E-Commerce Clothing Store

A full-stack e-commerce application built with Next.js, Prisma, PostgreSQL, and TypeScript.

## Goal

Build a production-ready clothing store from scratch while learning backend engineering, database design, authentication, payments, and deployment.

The project focuses on understanding the architecture rather than simply building features.

---

## Tech Stack

Frontend
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

Backend
- Next.js Server Actions

Database
- PostgreSQL

ORM
- Prisma

Authentication
- Email/Password
- Google OAuth (future)

Deployment
- Vercel
- Neon PostgreSQL
